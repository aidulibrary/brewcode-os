# 社区方案 · Community Recipes

把你的 `.brew.json` 文件放入此目录，提交 PR 即可贡献方案。

每个方案会自动通过 [brew.schema.json](../../packages/standards/brew.schema.json) 校验，
通过后出现在 [BrewRepo](https://repo.礼字号.中国) 的「社区方案」分区。

## 三步贡献

1. Fork → 创建 `.brew.json` 文件放入此目录
2. 发起 Pull Request → CI 自动校验
3. 审核通过 → 合并后上线

详见 [贡献指南](../../CONTRIBUTING.md)。
