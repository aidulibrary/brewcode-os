## A. 核心参数映射

| 序号 | .brew 字段 | 设备参数名 | 映射方式 | 设备误差 |
|------|-----------|-----------|---------|---------|
| 1 | `recipe.dose.value` | `target_weight` | 直接 / Direct | ±0.3g |
| 2 | `recipe.waterAmount.value` | `target_vol` | 直接 / Direct | ±3ml |
| 3 | `recipe.waterTemperature.value` | `target_temp` | 直接 / Direct | ±1.0°C |
| 4 | `recipe.grindSize.value` | `grind_dial` | 转换 / Convert | ±1 刻度 |

### 研磨度刻度 → 微米值对应关系（基于 D1 device_registry 真实数据）

| 研磨刻度 | 微米值（μm） | 适用场景 |
|:---:|:---------:|------|
| 5 | 400 | 细研磨，土耳其咖啡 |
| 10 | 700 | 中研磨，手冲通用 |
| 15 | 1100 | 粗研磨，法压壶 |

> 转换公式：`grind_dial = grindSize(μm) × (15 / 1100)`，即每刻度约等于 73μm。EK43 无级微调研磨，可在 1-15 范围内精细调节。

### EK43 刻度 10 = 700μm 填写示例

```json
{
  "recipe": {
    "dose": { "value": 15.0, "unit": "g" },
    "waterAmount": { "value": 250, "unit": "ml" },
    "waterTemperature": { "value": 93, "unit": "°C" },
    "grindSize": { "value": 700, "unit": "μm" }
  }
}
```

当 `.brew` 文件中 `grindSize.value = 700μm` 时，设备应显示 **EK43 刻度 10**，实现手冲通用中研磨的标准参数映射。
