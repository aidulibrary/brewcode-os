开源标准

.brew 是一种开放的 JSON 文件格式。任何人都可以阅读、创建、分享。零专利，零围墙。

◈

数据主权

你的冲煮记录、味觉偏好、设备参数属于你。我们不拥有你的数据，不售卖，不做广告。

⑂

社区驱动

改进从 Fork 开始，共识通过 Pull Request 达成。没有人独自拥有这个标准。

开始冲煮

打开 BrewPlayer，载入 .brew 文件，跟着步骤冲一杯好咖啡。

种子方案库

精选冲煮方案，覆盖主流器具、产区、烘焙度。每个方案都是一个 .brew 文件。

加入我们

BrewCode OS 是一个开放项目。标准、工具、方案——全部开源，全部在 GitHub 上协作。
