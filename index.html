BrewCode OS

The Universal Language
for Coffee Brewing

One file format. Every device. Any barista.

BrewCode OS is an open-source project that establishes a universal digital protocol for coffee brewing. At its heart is the .brew file — a JSON format that describes every parameter of a brew recipe: dose, water temperature, grind size, and step-by-step pouring instructions. A single .brew file can be parsed, executed, and reproduced on any compatible device, anywhere in the world.

Six-Layer Architecture

1. Standard

.brew JSON Schema (CC0) — the core protocol, released into the public domain

2. Tools

BrewPlayer, BrewRepo, and BrewForge — three open-source applications for playing, browsing, and creating brew recipes

3. AI Services

Flavor diagnosis, recipe generation, and semantic translation — edge computing via Cloudflare Workers

4. Data

100 seed recipes covering 20 origins, 6 brew methods, and recipes signed by world champions

5. Infrastructure

Cloudflare Pages + Workers + D1 + R2 — zero-cost global infrastructure

6. Certification

BrewCode Compatible — an interoperability certification for third-party brewing hardware

BrewCode Compatible Certification

Hardware manufacturers can certify their devices for interoperability with the .brew ecosystem. Three levels are available: L1 Ready (free, parameter reception), L2 Interactive ($99/year, step-by-step execution), and L3 Native ($299/year, full data recording). Certified devices appear in the public directory and may display the BrewCode Compatible mark on product packaging and websites.

Philosophy

Flavor-Parameter Isomorphism. Any describable flavor difference corresponds to a measurable parameter difference. Coffee is physics, not mysticism.

Knowledge as a Public Good. Brewing knowledge belongs to everyone. The core standard is CC0, and all code is MIT — forever.

Data Sovereignty. Your brewing data is yours. BrewCode OS collects nothing, owns nothing, tracks nothing.
